Website 7

Japanese English Dictionary

Fully functional dictionary app with complex regex search for Japanese and English, double wildcard, and live search results. Also has a flash card system to randomly pull from dictionary or a user chosen system for their own word collection. Features all latest security practices and MVC controller model.

Tech Stack

Features

  • Live form validation with color and text indicators.
  • Live search that works with arrow key and enter.
  • Complex regex search in Japanese and English plus allowing for double wildcard queries.
  • 5 layer user form validation.
  • All latest security measures and anti-hacking features implemented.
  • Random word flashcards from dictionary and custom user flashcard system to save words in 4 categories in personal collection based on mastery level. Each level has a different random chance of being drawn. More functionality to come.
  • Full live admin CRUD with self-populating forms and beautiful user interface.
  • Use of latest CSS and HTML tricks to reduce complex javascript, CDNs and other dependencies.
  • Complex usage cases of EJS templates to dynamically update user interface based on user selection. Also has look and feel of a SPA even thought it does not boast full DOM control like REACT.

Experience Gained

  • Learned through much pain the many ways to connect the front and back end through params, AJAX calls, Axios, queries, and of course req.body.
  • Learned EJS very well with very complex templates.
  • Learned how to code with both English and Japanese using Wanakana library and other tools to give expected functionality. Backend schema all in Japanese.
  • Learned in and out how to make an MVC controller with best practices for optimal performance, DRYness, scalability and of course ease of understanding and updates for programmers. Well documented and read-it-and-understand-it variables.
  • Learned how to implement all of the latest security techniques. Still plan to add further authentication with email registration, two-factor authentication based on IP and Google Catcha for an anti-bot countermeasure.
  • Learned how to make complex conditionals and regex to validate and perform database queries.
  • Learned the groundwork for creating very complex e-commerce/graph-like schemas even though I currently only link words to a user collection.
